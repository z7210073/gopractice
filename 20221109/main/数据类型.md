# 数据类型

## 基本数据类型

- 数值型
   - 整数型(int,int8,int16,int32,int64,uint,uint8,uint16,uint32,uint64,byte)
   - 浮点型（float32，float64)
   - 字符型（没有单字符型（char），使用byte来保存单字符）
   - 布尔型（bool)
   - 字符串(string)

- 复杂/派生数据类型
   - 指针（pointer)
   - 数组
   - 结构体(struct)
   - 管道(Channel)
   - 函数
   - 切片（slice）
   - 接口（interface）
   - map
